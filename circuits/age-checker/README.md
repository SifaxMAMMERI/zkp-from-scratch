# Age Checker Circuit

This circuit uses the `GreaterEqThan` comparator to verify if a private `age` is greater than or equal to a public `limit`.

*Private Input**: `age` (The user's actual age)
*Public Input**: `limit` (The required age, e.g., 18 or 21)
*Output**: `isAdult` (Returns 1 if age >= limit, 0 otherwise)
